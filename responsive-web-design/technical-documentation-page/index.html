Introduction

The Bash Shell (Bourne Again SHell) is a command-line interface (CLI). This is an introduction to basic Bash commands.

List directory contents (ls)

ls command is used to view all the files within the specified directory.

Syntax:

ls [option(s)] [file(s)]

Common options:

  • -a (all files and folders, including ones that are hidden and start with a .)
  • -l (list in long format)

Change directory (cd)

cd command is used to change the current working directory.

Syntax:

cd directory_name

Create a directory (mkdir)

mkdir command is used to create directories.

Syntax:

mkdir [option(s)] directory_name(s)

Common options:

  • -p (a flag which enables the command to create parent directories as necessary)
  • -v (displays a message for every directory created)

Copy files & directories (cp)

cp command is used when you need to back up your files.

Syntax:

cp [option(s)] current_name new_name

Common options:

  • -r (copying directory structure)
  • -b (creates the backup of the destination file in the same folder with the different name and in different format)

Creates a file (touch)

touch command is used to create new files.

Syntax:

touch [option(s)] file_name(s)

Common options:

  • -a (change the access time only)
  • -m (change the modification time only)
  • -r (use the access and modification times of file)
  • -d (update the access and modification times)

Prints text to terminal (echo)

echo command is used to print text to the terminal window and is typically used in shell scripts and batch files to output status text to the screen or a computer file.

Syntax:

echo [option(s)] [string(s)]

Common options:

  • -e (enable interpretation of backslash escapes)
  • -n (do not print the trailing newline)

Move or rename directory (mv)

mv command is used to move or rename directories.

Syntax:

mv [option(s)] argument(s)

Common options:

  • -f (force move them and overwrite files without checking with the user)
  • -i (prompt confirmation before overwriting files)

Remove files (rm)

rm command is used to remove files.

Syntax:

rm [option(s)] file_name

Common options:

  • -i (prompt before every removal)

Exit out of a directory (exit)

The exit command is used to close a terminal window, end the execution of a shell script, or log you out of an SSH remote access session.

Syntax:

exit

Clear your terminal (clear)

clear command is used to clear all previous commands and output from consoles and terminal windows.

Syntax:

clear
